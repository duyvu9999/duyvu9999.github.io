== Description ==

Elementor page builder addon to display posts in a grid. Useful for generating post grid from your blog posts with multiple options.

[View Demo](https://wpconcern.com/demo/post-grid-elementor-addon/) | [View Details](https://wpconcern.com/plugins/post-grid-elementor-addon/) | [Support](https://wordpress.org/support/plugin/post-grid-elementor-addon/) | [Buy Pro](https://wpconcern.com/plugins/post-grid-elementor-addon/)

== Installation ==

1. Go to Plugins > Add New and search for Post Grid Elementor Addon , install and activate it.
2. We are good to go. No configuration needed.
3. Post Grid is added under WPC Elements on Elementor page customization panel.


== Frequently Asked Questions ==

= What does this plugin needs to work ? =

This is an addon of Elementor page builder plugin so this plugin works if Elementor is installed and activated.

= Do this plugin works with all themes ? =

Yes, this plugin works with all themes.

= Can I add multiple layouts using this plugin ? =

Yes, it comes with 5 layouts options. Additionally, you can use options available to create extra layouts as well.
